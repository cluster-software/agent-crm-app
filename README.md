# Agent CRM Desktop App Releases

This repository hosts Agent CRM desktop app release assets for download and
auto-update compatibility.

## Current Locations

- Desktop downloads: https://github.com/cluster-software/agent-crm-app/releases

## Release Feed

Do not build new DMGs from this repository. New desktop app builds are published
to this repository's GitHub Releases feed so existing installed apps can
continue to update from the same location.
